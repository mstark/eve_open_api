## Installation

### Build a gem

To build the Ruby code into a gem:

```shell
gem build eve_open_api.gemspec
```

Then either install the gem locally:

```shell
gem install ./eve_open_api-0.2.6.dev1.gem
```
(for development, run `gem install --dev ./eve_open_api-0.2.6.dev1.gem` to install the development dependencies)

or publish the gem to a gem hosting service, e.g. [RubyGems](https://rubygems.org/).

Finally add this to the Gemfile:

    gem 'eve_open_api', '~> 0.2.6.dev1'

### Install from Git

If the Ruby gem is hosted at a git repository: https://github.com/mstark/eve_open_api, then add the following in the Gemfile:

    gem 'eve_open_api', :git => 'https://github.com/mstark/eve_open_api.git'

### Include the Ruby code directly

Include the Ruby code directly using `-I` as follows:

```shell
ruby -Ilib script.rb
```
